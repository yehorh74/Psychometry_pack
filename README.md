# Psychometry_pack
Biblioteka stworzona dla przeprowadzenia podstawowej analizy psychometrycznej danych kwestionariuszowych. Pakiet oferuje narzędzia do pracy z klasyczną teorią testów (CTT), umożliwiając badaczom i psychologom precyzyjną ewaluację narzędzi pomiarowych.
Biblioteka zawiera narzędzia do sprawdzania struktury testów:
  EFA (Exploratory Factor Analysis): Eksploracyjna analiza czynnikowa służąca do wykrywania liczby wymiarów w danych.
  Rotacje: Implementacja rotacji (np. Varimax, Oblimin) ułatwiająca interpretację ładunków czynnikowych.
Klasyczna Teoria Testów (CTT):
  Współczynnik rzetelności Alfa Cronbacha.
  Analiza trudności pozycji (p-values) oraz mocy dyskryminacyjnej (korelacja pozycja-wynik).
  Korelacja r-Pearson i rho-Spearmana dla analizy trafności.
Oraz opcje rekodowania i pandas-friendly metody do obliczania wyników testów i walidacji danych.
